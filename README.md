# new-webapp

> Starter project for Webpack SPA webapps.


## Running locally for development

```bash
./scripts/develop.sh
```


## Building

### Compiling all assets

```bash
./scripts/compile.sh
```

### Packaging for deployment

```bash
./scripts/package.sh
```


## Linting source code

```bash
./scripts/lint.sh

# Automatically fix certain linter errors
./scripts/lint.sh --fix
```


## Running unit tests

```bash
./scripts/test.sh

# Run in watch mode
./scripts/test.sh --watchAll
```
